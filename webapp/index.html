How it Works: Vault-Grade Privacy

1. No Network Connection

This app uses a strict Content Security Policy that disables all network requests (`connect-src 'none'`). Even if the app wanted to send your data, the browser would block it.

2. WebAssembly (WASM) Isolation

The PDF parsing engine is a Rust-compiled binary that runs in a separate Web Worker. It exists entirely in your device's memory.

3. Zero Persistence

Your transactions are never saved to disk. When you refresh the page or click 'Clear Session', all parsed data is permanently wiped from memory.

4. Open Source Audit

The logic is transparent and can be audited by any security professional. We prioritize privacy over everything.
